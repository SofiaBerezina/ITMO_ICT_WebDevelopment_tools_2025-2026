## 📋 О проекте

Тайм-менеджер - серверное приложение для управления задачами и временем.

### Технологии

- **FastAPI** - веб-фреймворк
- **SQLModel** - ORM
- **PostgreSQL** - база данных
- **Alembic** - миграции
- **JWT** - аутентификация
- **bcrypt** - хэширование
